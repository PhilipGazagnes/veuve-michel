## Features

This boilerplate has :

- Nuxt 3
- Tailwind CSS
- TS
- Prettier / ESLint
- Device detect

## Setup

Make sure to install dependencies:

```bash
# yarn
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# yarn
yarn dev
```

## Production

Build the application for production:

```bash
# yarn
yarn build
```

Locally preview production build:

```bash
yarn preview
```
